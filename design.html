Summary

Integrate vulnerabilities reporting into CI Scan and create a modern Dashboard with Vulnerability, Code Quality metrics as well as App Risk health.

CURRENT STATE

App team's have to login to multiple data sources to get metrics around vulnerabilities and code quality. Time consuming and difficult to understand.Centralized view of security metrics is not available.

TARGET STATE

Integrate Vulnerability data reporting into CI Build scan, Shift left early. Alert App-Dev partners regarding existing vulnerabilities during the Build phase. Use Olympos Risk dashboard that show's metrics around Vulnerabilities and Code Quality.

HOW DO WE GET THERE?

Gather data from the ESP(Enterprise Security pipeline), Checkmarx and CxAdmin tools. Filter key important data from the information that we captured from above tools. Create high-level metrics dashboards that help's app-dev partners to visualize the data with respect to security and code- quality.

WHAT DO WE NEED TO AUTOMATE

Create service accounts against ESP, Checkmarx and CxAdmin Tools. Need programmatic access to ESP, Checkmarx and CXAdmin tools.
